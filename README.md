## 🧭 Como Usar
### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado (versão LTS recomendada).

### Rodando um exemplo
Na pasta `Codigos/Exemplo_Aula_01`:
```bash
node hello_world.js
```

### Sugerido (opcional)
- Padronização de código com ESLint e Prettier.
- Commits semânticos (ex.: `feat: adiciona exemplo de funções`).
